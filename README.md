# DailyMate

A collection of daily quiz apps that help you learn through LLM-generated questions and evaluations.

## Apps

| App | Topic | Status |
|-----|-------|--------|
| [TorchMate](torchmate/) | PyTorch internals & best practices | Live |
| [EnglishMate](englishmate/) | English vocabulary & grammar | In progress |

## Getting Started

Each app is a standalone Next.js project. Navigate into the app directory and follow its README:

```bash
cd torchmate   # or englishmate
npm install
npm run dev
```
